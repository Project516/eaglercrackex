# Eaglercraftex

A modern fork of Eaglercrack with additional features, improvements, and a refreshed web interface.

## Features
- Clean, modern, and responsive website UI
- Easy navigation for clients, servers, packs, and worlds
- Consistent branding and accessibility improvements
- Blog and update sections
- Community Discord integration

## Project Structure
- `index.html` — Home page
- `clients.html`, `servers.html`, `packs.html`, `worlds.html` — Main sections
- `more/` — FAQ, forms, partners, updates, and more
- `blogs/` — Blog posts and templates
- `assets/` — Images, icons, and resource packs
- `style.css` — Main stylesheet
